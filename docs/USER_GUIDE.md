# 积存金决策辅助系统使用说明

## 系统用途

这个系统用于观察黄金价格、比较国内金价与伦敦金折算价、回测规则策略，并给出仓位层面的辅助判断。它不是自动交易系统，也不构成投资建议。

## 启动方式

```powershell
.\.venv\Scripts\Activate.ps1
streamlit run app.py
```

打开页面后访问 `http://localhost:8501`。

## 页面怎么用

1. 选择数据源。
   - `AKShare/上金所`：默认推荐，用上金所历史日线做回测。
   - `演示数据`：没有网络时用于试跑。
   - `CSV`：上传自己的历史价格。

2. 核对当前价格。
   - `国内价(元/克)`：用于策略计算的最新国内价格。
   - `伦敦折算`：伦敦金 XAU/USD 按 USD/CNY 折成人民币/克。
   - `国内溢价`：国内价相对伦敦折算价的差异。

页面中的九百多元不是系统凭空换算出来的单一价格。`国内价(元/克)` 来自上金所延时价或历史日线；`伦敦折算` 才是由 XAU/USD 和 USD/CNY 换算出来的价格。

3. 使用换算工具。
   - `伦敦金 -> 元/克`：输入或使用自动带入的 XAU/USD 与 USD/CNY，换算成人民币/克。
   - `元/克 -> 伦敦金`：输入国内或银行金价，按 USD/CNY 反推等价 XAU/USD。
   - 换算工具只做单位和汇率换算，不会覆盖策略使用的行情数据。

4. 设置自动刷新。
   - `自动刷新` 可以选择关闭、30 秒、1 分钟、5 分钟或 15 分钟。
   - 自动刷新使用 Streamlit 局部刷新，不会刷新整个浏览器页面，也不会清空侧边栏参数。
   - 自动刷新只代表系统会定时重新请求公开行情源；上金所公开页面本身仍是延时行情，不是实时 tick 行情。

5. 调整策略参数。
   - `初始资金`：回测假设的起始资金。
   - `买卖价差 bps`：模拟买卖报价差或交易摩擦。
   - `短均线`、`长均线`：判断趋势。
   - `深度回撤`：从阶段高点回落多少才算明显回撤。
   - `调仓阈值`：目标仓位与实际仓位差异超过多少才交易。

6. 查看结果。
   - `当前信号`：策略对当前状态的判断。
   - `目标仓位`：策略建议的黄金资产比例。
   - `策略累计收益`、`最大回撤`：回测表现。
   - `价格与均线`：价格趋势和均线位置。
   - `资金曲线`：策略与买入持有基准对比。
   - `最近调仓`：最近发生的模拟买入、卖出记录。

## 关键名词

`积存金`：银行提供的黄金积存产品，通常按元/克报价，可以主动买入、定投、赎回或兑换实物，具体规则以银行为准。

`Au99.99`：上海黄金交易所黄金现货合约，成色不低于 99.99%，报价单位是元/克。

`伦敦金 / XAU/USD`：国际现货黄金常用报价，单位是美元/金衡盎司。它是国际参考价，但不是国内银行积存金的直接成交价。

`金衡盎司`：贵金属计量单位，1 金衡盎司 = 31.1034768 克。

`伦敦金折人民币`：计算公式是 `XAU/USD * USD/CNY / 31.1034768`，结果单位是元/克。

`国内溢价`：`国内价 / 伦敦折人民币 - 1`。正数表示国内价格高于伦敦折算价，负数表示低于伦敦折算价。

`买卖价差 bps`：bps 是基点，1 bps = 0.01%。35 bps 表示约 0.35% 的交易摩擦。

`均线 MA`：过去若干交易日的平均价格。短均线高于长均线通常代表趋势偏强。

`RSI`：相对强弱指标，用来观察价格是否过热或降温。

`回撤`：价格从阶段高点下跌的幅度。

`目标仓位`：策略希望黄金资产占总资产的比例。例如 35% 表示 10 万元资金中约 3.5 万元配置黄金。

`最大回撤`：回测资金曲线从历史高点到低点的最大跌幅。

`夏普比率`：收益相对波动的粗略衡量，越高通常代表单位风险收益越好。

## 当前信号含义

`积极买入`：趋势仍偏强，且出现较深回撤和动量降温。

`小额买入`：趋势偏强，价格没有明显过热。

`试探买入`：趋势不强，但回撤较深，只适合小比例分批。

`持有`：趋势仍在，但性价比不够明确。

`减仓`：价格可能偏热，策略倾向降低仓位。

`防守观望`：趋势偏弱，策略倾向低仓位。

## 数据来源和更新

国内历史日线在 `src/gold_advisor/data.py` 的 `load_sge_prices()` 中通过 AKShare 获取。

上金所延时行情在 `src/gold_advisor/data.py` 的 `get_sge_delayed_quote()` 中解析上海黄金交易所官方页面。

伦敦金和美元兑人民币在 `src/gold_advisor/market.py` 的 `get_london_gold_conversion()` 中获取。当前 MVP 使用 Stooq CSV 接口读取 `XAUUSD` 和 `USDCNY`，再调用 `convert_london_gold_to_cny_per_gram()` 折算。

如果接口失败，页面会显示不可用原因；换算工具在拿到自动行情后会默认带入最新 XAU/USD、USD/CNY 和国内参考价。

## 自动刷新说明

系统的自动刷新是“页面内局部刷新”，不是浏览器整页刷新。它会定时重新渲染行情、折算价、策略信号和图表区域，侧边栏输入值会保留。

刷新间隔也会影响行情缓存。比如选择 `1 分钟`，系统每分钟会尝试重新获取上金所延时价、XAU/USD 和 USD/CNY；历史日线仍按较长缓存处理，因为日线不需要高频请求。

## 注意事项

伦敦金、上金所价格和银行积存金价格不是同一个东西。银行报价可能包含点差、手续费、库存、交易时段和风控调整。当前系统没有接入某家银行的私有报价接口，因此主行情使用国内公开参考价，银行实际买卖价仍以银行 App 为准。

国际市场常把 LBMA Gold Price 或 XAU/USD 作为重要参考，但国内积存金不能简单等同于伦敦金折人民币。更合理的做法是同时看国内报价、伦敦折算价和两者价差。

当前系统是研究工具，不会自动下单。所有策略结果都应先和无脑定投、买入持有、空仓等基准比较后再参考。
